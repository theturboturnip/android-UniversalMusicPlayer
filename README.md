Turnip Music Player
=====================================

This is a music player forked from the Google Universal Music Player sample.


Pre-requisites
--------------

- Android SDK v17

Getting Started
---------------

This uses the Gradle build system. To build this project, use the
"gradlew build" command or use "Import Project" in Android Studio.
